Text Hiding Using Audio Steganography


Using steganography hiding the message in the audio file in such a way, that there would be no perceivable changes in the audio file after the message insertion. It could be used as a secret method of communication to exchange important information especially in warfare.


Covid Pneumonia Detection Using Deep Learning


Detecting covid pneumonia by analyzing chest x-ray images by using DNN and VGG16 architecture model to pre-process the data, feature extraction and also to perform training and testing of images. Then uses Random forest algorithm in order to classify images i.e whether it is covid-19 or not based on the model.
